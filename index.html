JATZ Mail

JATZ Mail is an open source email sender. Using JATZMAIL you can send emails to anyone. It is completely free to download and use, and new updates are coming out
constantly. JATZ Mail can be dowloaded in .pyc form or .exe file type. We are working on making a .py format available. You can always submit feedback
for JATZ Mail because we want to let you have the best experience possible. To submit feedback, download the latest version of JATZ Mail from out
Github Repository and press the Rate JATZ Mail button.

Our Github Repository

Using our Github repository you can download the latest version of JATZ Mail, or even an old one! Starting from JATZ Mail version indev 0.3, a .py file is not available for
download.We are working on using minimum private information in the .py file, so for now we have to provide either a compiled or executable document.

How to use JATZ Mail?

JATZ Mail is a very user friendly program. As soon as you launch JATZ Mail, you will see five entry boxes. The top one is for extra features. To test it out type iamdev
into the top entry box. You will see a new section pop up. This is the "Dev Section" It has some extra features. New ones are coming out in indev 0.4

The other four entry boxes are for sending the email. The first one is for the message you want to send. The second one is for the email you want to send the
message to. The third entry box is for the password to the email you want to send the message from. This is required since you will not be able to log into your
email account without it. The last entry box is for the email you want to send from. This should be the same email to which you provided the password for.
